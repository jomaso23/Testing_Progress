Prueba Unitaria: testea una sola funcionalidad.
Suite de testeo: Es un conjunto de pruebas que testea todo el proyecto.
Cobertura - Coverage:  % de lineas de código estamos testeando.

El usuario programador no crea las pruebas, solo entrega un software que
    compila.

Usuario Testing
QC: Quality Check       (Pruebas funcionales - Caja Negra)
QA: Quality Assurance   (Pruebas de toda la arquitectura, 
                            automatiza pruebas - Caja Blanca)
